Quick Install
-------------
1. Choose the correct add-in:
   - Use RKExcelReportCompare64-packed.xll for 64-bit Excel.
   - Use RKExcelReportCompare-packed.xll for 32-bit Excel.
2. If Windows blocks the file, right-click the .xll file, choose Properties, select Unblock if shown, and click OK.
3. Open Excel.
4. Go to File -> Options -> Add-ins.
5. Set Manage to Excel Add-ins and click Go.
6. Click Browse and select the .xll file.
7. Make sure the add-in is checked.
8. Open the RK Compare tab in Excel.

For full instructions, open USER_GUIDE.html.
